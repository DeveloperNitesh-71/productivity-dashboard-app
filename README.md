# Productivity Dashboard App 🚧 (Work In Progress)

A modern productivity dashboard currently under development to help users manage tasks, track focus sessions, and monitor performance.

## 🚀 Current Features

- ✅ Task Management (Add / Delete / Complete Tasks / Edit Tasks)
- ⏱ Focus Timer (Minute & Second Tracking)
- 📆 Calender
- 🕛 Clock
- 💾 Local Storage Persistence
- 📱 Responsive Layout
-  Drag and drop feature
-  and much more....

## 🔄 Upcoming Features

- 📊 Productivity Analytics Section
- 🗂 Task Categories
- 🌙 Dark / Light Mode Toggle
- 📈 Weekly Performance Tracking
- 🔐 User Authentication (Future Upgrade)

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage API

## 🎯 Project Goal

This project is being built to strengthen real-world frontend development skills including:

- DOM manipulation
- Timer logic implementation
- State management
- UI structuring for dashboard-style applications
- Writing clean, scalable JavaScript code
